# Selenium E-Commerce Automation Framework

## Project Overview
This project is a Selenium Web Automation Framework developed using Java, Selenium WebDriver, TestNG, Maven, and Page Object Model (POM) design pattern.

The framework automates login functionality of an e-commerce application and follows industry-standard automation practices.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Git
- GitHub

## Features
- Browser initialization using Selenium WebDriver
- Page Object Model implementation
- Login automation test
- TestNG assertions
- Maven dependency management
- Reusable framework structure

## Test Scenario
1. Launch application
2. Enter valid username
3. Enter valid password
4. Click Login button
5. Verify successful login
